    What is Anysols?

    Sleek and powerful platform for faster and easier application development. Anysols empowers developers to build applications that live on the web platform.

    Quick start

    Install the latest version of Anysols globally with npm:

    npm install -g anysols

    Create a new project:

    mkdir new-project
new-project
anysols setup

    Run the project

    anysols run
